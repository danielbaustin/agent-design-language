## Metadata

## Status

Forward planning. `v0.94.1` is not yet an active implementation milestone and
has a candidate issue-wave package but no final opened issue wave.

## Purpose

`v0.94.1` is the planned payments, settlement, and economic-agency milestone.

It should give the previously deferred economic rails a dedicated home after
governance/security (`v0.93`) and secure execution/trust convergence (`v0.94`)
are in place.

## Milestone Role

`v0.94.1` should establish:

- payment adapter boundary
- `x402` and Lightning adapter planning surfaces
- settlement rules
- economic accounting and trace events
- economic-agency and performance-position surfaces
- bounded payment/economic demos

## Scope Summary

### In scope

- payments and settlement architecture
- `x402` and Lightning adapter boundaries
- accounting, ledger, and economic trace surfaces
- economic-agency follow-on and demos

### Out of scope

- silently pulling payments work into `v0.93` or `v0.94`
- claiming production financial compliance or live monetary deployment
