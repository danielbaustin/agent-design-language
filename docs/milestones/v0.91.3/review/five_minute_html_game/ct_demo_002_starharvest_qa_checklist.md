# Starharvest QA Checklist

## Run Path

- [ ] HTML file opens cleanly in a browser
- [ ] helper command prints a valid local URL

## Keyboard Control

- [ ] `WASD` or arrow keys move the skiff
- [ ] `Shift` boosts movement
- [ ] `Space` interacts with the nearest asteroid in range
- [ ] `R` restarts the run

## Gameplay Loop

- [ ] empty asteroids can be planted
- [ ] growing asteroids can be tended
- [ ] ripe asteroids can be harvested
- [ ] score increases on harvest
- [ ] seeds change over time

## Win/loss

- [ ] run can end in a successful target hit
- [ ] run can end when timer expires
- [ ] run outcome panel updates truthfully

## Visual/UX

- [ ] HUD values remain readable
- [ ] selected asteroid panel updates
- [ ] upgrades can be purchased when affordable
- [ ] hazards visibly move through the playfield
