# Release Notes - v0.90.5

## Status

Final release notes for `v0.90.5`.

The implementation wave, feature-proof coverage, quality gate, docs/review
convergence, internal review, third-party review, accepted-finding disposition,
next-milestone handoff, and release ceremony are complete.

## Headline

`v0.90.5` delivers Governed Tools v1.0 for ADL: portable tool descriptions,
authority-bound execution contracts, deterministic compilation, policy and
Freedom Gate mediation, governed execution, redacted trace evidence, bounded
model evaluation, the flagship governed-tools proof packet, and the first
landed ACIP / Comms tranche.

## Highlights

- Universal Tool Schema (`UTS`) public-compatible schema and conformance
- ADL Capability Contracts (`ACC`) for authority, privacy, visibility,
  delegation, trace, and replay boundaries
- deterministic `UTS` to `ACC` compiler
- policy injection and Freedom Gate mediation before execution
- governed executor for approved ACC-backed actions
- trace, replay, and redacted evidence surfaces
- dangerous negative suite that fails closed
- bounded model-proposal benchmark and local / Gemma evaluation
- Governed Tools v1.0 flagship demo
- explicit feature-proof coverage before review convergence
- first landed ACIP / Comms tranche with canonical envelope, identity,
  invocation, review/coding specializations, and proof coverage
- ADR 0015 for governed-tools execution authority architecture
- clean handoff into `v0.91` and `v0.91.1`

## Non-Claims

- `v0.90.5` does not claim that `UTS` alone grants execution authority.
- `v0.90.5` does not permit arbitrary shell, network, or secret-bearing
  execution because a model emitted valid JSON.
- `v0.90.5` does not ship payment rails, billing, legal contracting, or
  inter-polis economics.
- `v0.90.5` does not ship the full `v0.91` moral/cognitive-being substrate.
- `v0.90.5` does not ship the full `v0.91.1` adjacent-systems package or the
  full Gemma/local/remote comparison report.

## Release Note Rule

This final note keeps only landed evidence and explicit later-band placement.
