# v0.95 Feature: Aptitude Atlas Model-Evaluation Platform

## Status

Forward-planning feature contract for `v0.95`.

## Purpose

Give Aptitude Atlas a canonical tracked feature home as the ADL-powered model
capability and aptitude evaluation product lane, beyond the initial harness and
comparison work already landed earlier.

## Source Inputs

- `docs/planning/ADL_FEATURE_LIST.md`
- `docs/GLOSSARY.md`
- `docs/milestones/v0.90.1/ideas/ANRM_SCAFFOLDED_SMALL_MODEL_EXPERIMENT.md`
- `docs/milestones/v0.85/features/ROAD_TO_v0.95.md`

## Scope

This feature should establish:

- the productized evaluation lane beyond one-off benchmark scorecards
- relationship between capability harnesses, aptitude families, and public or
  reviewer-facing reporting
- a bounded MVP surface for model comparison that remains evidence-backed

## Non-goals

- a vague leaderboard without reviewable methodology
- collapsing all cognitive or provider quality into one mythology score
- claiming broad product maturity from the `v0.91.1` harness alone

## Completion Target

`v0.95`
