# Vision - v0.90.1

## One Sentence

Build the first reviewable Runtime v2 foundation prototype: a bounded manifold
with provisional citizens, kernel services, snapshot/rehydration, invariant
evidence, and operator controls, delivered through a front-loaded compressed
execution lane.

## Why This Milestone Exists

v0.90 proves long-lived runtime supervision. v0.90.1 should take the next
small step: show that a persistent world substrate can exist without claiming
full personhood, moral civilization, or the first true birthday.

The point is not spectacle. The point is a concrete, inspectable runtime
foundation that later v0.91 and v0.92 work can safely inherit.

This milestone also proves that compression can be an operating condition rather
than a release-tail wish: issue-wave generation, worktree-first safety, and
execution-policy truth should be in place before Runtime v2 coding starts.

## North Star

A reviewer should be able to run or inspect one bounded Runtime v2 prototype and
answer:

- What is the manifold?
- Which citizens exist, and why are they provisional?
- What kernel services keep the world coherent?
- What gets persisted before sleep?
- How does wake/rehydration preserve continuity without duplicating identity?
- What invariant failed, and where is the evidence?
- How can an operator pause, inspect, resume, or terminate the world?
- Did the milestone avoid preventable process drag while preserving review
  truth?

## What Success Feels Like

v0.90.1 succeeds when Runtime v2 stops being only a beautiful architecture
story and becomes a small executable substrate with visible boundaries.

It should also make the next milestone cheaper to run. v0.90.2 should inherit a
better issue-wave path, safer worktree discipline, and clearer compressed
validation rules.

It should feel like seeing the walls, gates, ledgers, clocks, and watchtowers of
the polis for the first time, with the road crew finally ahead of the caravan.
Not the full city. The foundation.

## What This Must Not Become

This milestone must not become a grab bag of identity, emotion, governance,
economics, and security dreams. Those matter, but they each need clean homes.

v0.91 remains the moral/emotional substrate band. v0.92 remains the first true
Gödel-agent birthday and identity/capability continuity band.
