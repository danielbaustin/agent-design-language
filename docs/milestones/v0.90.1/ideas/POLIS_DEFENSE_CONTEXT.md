# Polis Defense Context

If frontier models become very good at finding vulnerabilities quickly, the
polis needs its own defensive substrate. ADL should surface that insight early,
because a persistent cognitive world cannot be safe if its own runtime boundary
is opaque.

For v0.90.1 this means one bounded security-boundary proof:

- a normal runtime path receives an invalid action
- kernel or policy refuses it
- evidence is traceable
- the manifold remains coherent

This is not the full security ecology. It is the first wall stone.
