# v0.90.4 Feature Contracts

These feature docs define the implementation-facing contract-market substrate
for v0.90.4.

Read them with the milestone boundary in mind:

- v0.90.3 owns citizen state, standing, access control, projection, continuity,
  challenge, and appeal.
- v0.90.4 owns contracts, bids, evaluation, lifecycle authority, delegation,
  external counterparty participation, fixture-backed execution, and bounded
  review summaries.
- payment rails and full inter-polis economics remain later work.
