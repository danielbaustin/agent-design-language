# v0.94 Feature: Mental Time Travel / Temporal Self-Projection

## Status

Forward-planning feature contract for `v0.94`.

## Purpose

Define bounded mental time travel as temporal self-projection over remembered,
reconstructed, inferred, and simulated time.

## Source Inputs

- `docs/milestones/v0.94/README.md`
- `docs/milestones/v0.94/features/README.md`
- `docs/milestones/v0.94/features/SIGNED_TRACE_AND_TRACE_QUERY_v0.94.md`
- `docs/planning/ADL_FEATURE_LIST.md`

## Scope

This feature should establish:

- chronosense as prerequisite temporal self-location
- identity-bearing continuity across temporal frames
- memory as future-directed material
- inspectable counterfactual reasoning
- GHB-style bounded simulation
- distinction among observed, reconstructed, inferred, and simulated temporal
  content
- bounded temporal-self projection that remains evidence-aware rather than
  drifting into mythology or unconstrained narration

## Non-goals

- unconstrained “multiverse oracle” claims
- treating simulated branches as observed fact
- replacing signed trace, memory, or identity continuity with free-form
  time-travel metaphor
- claiming full Gödel-agent maturity from temporal rehearsal alone

## Completion Target

`v0.94`
