# Release Plan - v0.90.2

## Release Thesis

Release v0.90.2 only when Runtime v2 has completed the first bounded CSM run
and produced reviewable hardening evidence around that run.

## Required Release Evidence

- v0.90.1 inheritance audit
- compression readiness and issue-wave truth
- first CSM run packet contract
- manifold boot and citizen admission evidence
- governed episode and resource scheduling evidence
- Freedom Gate mediation evidence
- invalid action rejection evidence
- snapshot rehydrate wake continuity proof
- CSM Observatory packet and operator report
- invariant coverage map
- stable violation artifact contract and examples
- recovery versus quarantine proof
- duplicate activation, snapshot integrity, and replay-gap negative proofs
- bounded security-boundary hardening proof
- integrated first CSM run demo packet
- internal review and remediation record
- release notes that do not overclaim birthday, moral/emotional civilization,
  complete migration, or red/blue ecology

## Release Blockers

- Any claim that v0.90.2 is the first true Gödel-agent birthday
- Any unreviewed security-boundary overclaim
- Missing first CSM run proof packet
- Missing Observatory-visible run evidence
- Missing violation artifact proof
- Missing recovery/quarantine distinction
- Missing operator-review evidence
- Failing required CI without explicit release-owner disposition

## Handoff

The release must hand off cleanly to:

- v0.91 for affect, kindness, humor, morals, wellbeing, cultivation, and civic substrate
- v0.92 for first true birthday, identity continuity, capability rebinding, and richer memory
