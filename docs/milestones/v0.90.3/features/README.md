# v0.90.3 Feature Contracts

These feature contracts define the citizen-state substrate planned for v0.90.3.

They are implementation-facing planning docs. They do not claim shipped
behavior until the corresponding WPs land.

| Feature | Purpose |
| --- | --- |
| `CITIZEN_STATE_SECURITY_AND_FORMAT.md` | canonical private state, signed envelope, ledger, witnesses, receipts, and anti-equivocation |
| `STANDING_COMMUNICATION_AND_ACCESS.md` | citizen/guest standing, communication, inspection, and access-control boundaries |
| `SANCTUARY_QUARANTINE_AND_CHALLENGE.md` | conservative behavior and due-process artifacts for uncertain continuity |
| `LOCAL_SEALED_QUINTESSENCE_CHECKPOINTS.md` | local-first protected checkpoint strategy and backend seam |
| `ECONOMICS_PLACEMENT_BRIDGE.md` | narrow v0.90.3 resource-stewardship decision and v0.90.4 handoff |
