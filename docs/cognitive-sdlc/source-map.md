# C-SDLC Source Map And Supersession

## Status

Tracked source map for `#3120`.

## Purpose

This document records how local planning notes were promoted into the tracked
C-SDLC documentation home.

## Supersession Rule

The files in `docs/cognitive-sdlc/` are the canonical public/auditable C-SDLC
documentation set for current `v0.91.3` and `v0.91.4` planning.

The local `.adl/docs/TBD/cognitive-sdlc/` notes remain drafting history and
provenance. They are not the canonical source for current milestone execution.
