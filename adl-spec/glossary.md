# Glossary

- **ADL**: Agent Design Language.
- **Run**: A design-time binding of inputs to a workflow.
- **Session**: A runtime instance of executing a run.
- **Contract**: Declared requirements for tool inputs/outputs or step outputs.
- **Guard**: A validation/policy check enforcing a contract.
- **Normalization**: Deterministic text transformations applied before guards.
- **Repair**: A bounded retry strategy applied after a guard failure.
