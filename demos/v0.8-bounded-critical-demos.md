# v0.8 Bounded Critical Demos

This runbook provides reviewer-friendly commands for the milestone-critical bounded demos implemented in the `adl demo` runtime surface.

## Scope

Implemented now:
- bounded Godel + ObsMem stage-loop demo
- bounded multi-agent card pipeline demo
- bounded ObsMem retrieval demo

Deferred:
- autonomous mutation acceptance
- cross-workflow learning policy loops
- semantic retrieval ranking

## Commands

Run from repository root.

```bash
cargo run --manifest-path adl/Cargo.toml --bin adl -- demo demo-d-godel-obsmem-loop --run --trace --out ./out
cargo run --manifest-path adl/Cargo.toml --bin adl -- demo demo-e-multi-agent-card-pipeline --run --trace --out ./out
cargo run --manifest-path adl/Cargo.toml --bin adl -- demo demo-f-obsmem-retrieval --run --trace --out ./out
```
