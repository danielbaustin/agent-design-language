# Deep-Agents Comparative Wave Demo

Canonical command:

```bash
bash adl/tools/demo_v089_deep_agents_comparative_wave.sh
```

## What It Does

This demo deepens the earlier bounded comparative proof into a richer
multi-provider comparative packet.

ChatGPT, Claude, and Gemini each contribute one bounded position, and ADL then
writes a findings-first synthesis package that preserves one real disagreement
instead of flattening the comparative story.

## Why It Matters

The earlier proof row established the basic comparison.

This follow-on makes the story more interesting:

- multiple named perspectives
- visible tension instead of fake consensus
- a reviewer-friendly synthesis surface
- public positioning without benchmark-war theatrics
